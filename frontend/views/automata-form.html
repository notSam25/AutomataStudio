DFA and NFA

  • DFA transitions should be complete for every state and alphabet symbol
  • NFA transitions may branch to multiple destinations for the same symbol

PDA settings

  • Define the stack alphabet and initial stack symbol
  • Transitions can include a required stack symbol and a push symbol

Turing machine settings

  • Transitions can include a write symbol and head movement
  • Use L, R, or S for head movement

Graphical State Machine Editor

How to use:

  • Click on the canvas to add states
  • Click one state then another to create transitions
  • Click the same state twice to add a self transition
  • Double-click on a state to rename it
  • Right-click on a state or transition to delete it
